HTML Basics

Learn the fundamentals of HTML and start building your first web pages!

Lesson 1: Introduction to HTML

HTML stands for HyperText Markup Language. It is used to structure content on the web.

Lesson 2: HTML Tags and Elements

Learn about common HTML tags like <h1>, <p>, <a>, <img> and how to use them.

Lesson 3: Creating Your First Page

Combine tags to create a simple web page with headings, paragraphs, links, and images.

Lesson 4: HTML Lists and Tables

Learn how to structure information using ordered/unordered lists and tables.
